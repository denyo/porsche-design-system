## Accessibility support

### Keyboard

| Key / state                 | Function                                                                        |
| --------------------------- | ------------------------------------------------------------------------------- |
| `Tab`, `Shift-Tab`          | Moves focus to the scroll container or to next (or previous) focusable element. |
| `Arrow left`, `Arrow right` | Scrolls the container.                                                          |
